# 🖥️ SIPONTREN Backend (Sistem Informasi Pondok Pesantren)

Selamat datang di repository **Backend Sistem Informasi Pondok Pesantren Muhammadiyah Al-Amin Seputih Banyak - Lampung Tengah**.  
Backend ini berperan sebagai **RESTful API** yang menghubungkan frontend dengan database, serta menangani autentikasi, validasi data, dan manajemen file.

---

## 🚀 Fitur Utama

✅ **RESTful API Modular**  
> Menggunakan arsitektur Express JS dengan TypeScript yang mudah di-maintain.

✅ **Database MySQL + Prisma ORM**  
> Integrasi Prisma ORM untuk query database yang cepat dan aman.

✅ **Autentikasi & Otorisasi JWT**  
> Keamanan data menggunakan **JSON Web Token** + **bcrypt** untuk enkripsi password.

✅ **Upload File dengan Multer**  
> Mendukung upload file (gambar, dokumen) langsung dari frontend.

✅ **Validasi Input dengan Zod**  
> Semua data dari client divalidasi agar konsisten dan aman.

✅ **CORS & Middleware Modern**  
> Mendukung komunikasi aman antara frontend React dan backend.

✅ **Environment Variables**  
> Konfigurasi sensitif (DB, JWT_SECRET, dll) dikelola lewat `.env` dengan dotenv.

✅ **Status: Dalam Pengembangan**  
> Beberapa fitur masih dalam proses penyempurnaan.
